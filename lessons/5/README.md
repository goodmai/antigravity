# Урок 5: Типовые Скиллы (Skills) — Полный список (17/17)

Скиллы — это модульные пакеты, которые расширяют возможности агента. В папке `skills` находится ровно 17 инструментов. Ниже представлен полный каталог.

---

## 🎨 Визуальное искусство и дизайн

### 1. `algorithmic-art`

**Описание**: Создание генеративного искусства с помощью p5.js.

- **Промпт**: "Создай генеративное искусство на тему 'Биолюминесценция'".
- **Результат**: Интерактивный HTML с анимацией.
- **Юзеркейсы**: Визуальные инсталляции, динамические фоны.
- **Тесткейс**: Смена параметров через слайдеры меняет графику.

### 2. `canvas-design`

**Описание**: Создание статических постеров и арт-объектов (.png/.pdf).

- **Промпт**: "Создай постер 'Geometric Silence' для выставки".
- **Результат**: Файл .png высокого качества.
- **Юзеркейсы**: Печатная продукция, брендинг.
- **Тесткейс**: Проверка композиции и отсутствия наложений элементов.

### 3. `brand-guidelines`

**Описание**: Применение фирменного стиля Anthropic.

- **Промпт**: "Оформи мой документ в стиле Anthropic".
- **Результат**: Использование шрифтов Poppins/Lora и брендовых цветов.
- **Юзеркейсы**: Корпоративные отчеты.
- **Тесткейс**: Соответствие hex-кодов цветов брендбуку.

### 4. `frontend-design`

**Описание**: Создание высококлассных UI-интерфейсов без "ИИ-слопа".

- **Промпт**: "Спроектируй футуристичный дашборд для управления дронами".
- **Результат**: Профессиональный код (React/HTML/CSS) с уникальным дизайном.
- **Юзеркейсы**: Прототипирование сайтов, посадочные страницы.
- **Тесткейс**: Уникальность дизайна, использование адаптивной верстки.

### 5. `theme-factory`

**Описание**: Коллекция из 10 пресетных тем для стилизации артефактов.

- **Промпт**: "Примени тему 'Ocean Depths' к моей презентации".
- **Результат**: Полная смена цветовой схемы и шрифтовых пар.
- **Юзеркейсы**: Быстрое оформление слайдов и документов.
- **Тесткейс**: Проверка читаемости текста на выбранном фоне.

### 6. `slack-gif-creator`

**Описание**: Создание анимированных GIF, оптимизированных для Slack.

- **Промпт**: "Сделай GIF с пульсирующей звездой для эмодзи Slack".
- **Результат**: GIF 128x128 с низким весом.
- **Юзеркейсы**: Кастомные эмодзи, реакции в Slack.
- **Тесткейс**: Размер файла не превышает лимиты Slack для эмодзи.

## 📄 Документация и Офис

### 7. `doc-coauthoring`

**Описание**: Структурированный процесс совместного написания документации.

- **Промпт**: "Помоги мне написать техническое задание (spec)".
- **Результат**: Пошаговое руководство через сбор контекста и черновики.
- **Юзеркейсы**: Написание PRD, RFC, сложных инструкций.
- **Тесткейс**: Прохождение всех 3-х стадий процесса.

### 8. `docx`

**Описание**: Создание и редактирование документов Microsoft Word.

- **Промпт**: "Создай официальное письмо в .docx".
- **Результат**: Файл Word с поддержкой сложных стилей.
- **Юзеркейсы**: Юридические документы, письма, статьи.
- **Тесткейс**: Проверка структуры через `pandoc` или Word.

### 9. `pptx`

**Описание**: Создание презентаций Microsoft PowerPoint.

- **Промпт**: "Подготовь презентацию нового продукта в .pptx".
- **Результат**: Слайд-дека с заголовками, текстом и стилями.
- **Юзеркейсы**: Квартальные отчеты, питч-деки.
- **Тесткейс**: Открытие в PowerPoint и проверка макетов слайдов.

### 10. `xlsx`

**Описание**: Создание и редактирование таблиц Microsoft Excel.

- **Промпт**: "Создай таблицу бюджета в .xlsx с формулами суммы".
- **Результат**: Таблица с данными и функциональными ячейками.
- **Юзеркейсы**: Финансовый учет, списки данных.
- **Тесткейс**: Проверка работоспособности формул в Excel.

### 11. `pdf`

**Описание**: Манипуляция PDF: извлечение текста, слияние, разделение.

- **Промпт**: "Слей эти два PDF файла в один".
- **Результат**: Новый документ PDF.
- **Юзеркейсы**: Работа со сканами, сборка документов.
- **Тесткейс**: Проверка целостности итогового файла.

### 12. `internal-comms`

**Описание**: Ресурсы для внутренних коммуникаций (3P, новости, FAQ).

- **Промпт**: "Напиши еженедельный апдейт (3P) по проекту".
- **Результат**: Текст по формату Progress/Plans/Problems.
- **Юзеркейсы**: Командные рассылки, отчеты руководству.
- **Тесткейс**: Проверка соответствия структуры примерам из `examples/`.

## 🛠️ Разработка и Инструменты

### 13. `mcp-builder`

**Описание**: Руководство по созданию серверов Model Context Protocol.

- **Промпт**: "Создай MCP-сервер для работы с внешним API".
- **Результат**: Проект сервера с настроенными инструментами.
- **Юзеркейсы**: Интеграция агента с новыми сервисами.
- **Тесткейс**: Валидация через MCP Inspector.

### 14. `skill-creator`

**Описание**: Инструмент для разработки новых кастомных скиллов.

- **Промпт**: "Создай новый скилл для ревью кода".
- **Результат**: Полная структура папки скилла.
- **Юзеркейсы**: Упаковка новых знаний для агента.
- **Тесткейс**: Успешное выполнение `package_skill.py`.

### 15. `web-artifacts-builder`

**Описание**: Сборка сложных React/Tailwind/shadcn приложений в один файл.

- **Промпт**: "Собери дашборд аналитики на React".
- **Результат**: Один HTML-файл со всем функционалом.
- **Юзеркейсы**: Интерактивные инструменты в чате.
- **Тесткейс**: Проверка работы интерактивных элементов в браузере.

### 16. `webapp-testing`

**Описание**: Автоматизация тестирования сайтов через Playwright.

- **Промпт**: "Проверь работоспособность кнопок на главной странице".
- **Результат**: Скрипт тестирования и отчет.
- **Юзеркейсы**: Регулярные проверки качества (QA).
- **Тесткейс**: Успешное завершение теста без ошибок (exit code 0).

### 17. `inspect-artifacts`

**Описание**: Быстрый просмотр папки артефактов текущего разговора.

- **Промпт**: "Покажи список моих артефактов".
- **Результат**: Вывод содержимого директории `.gemini/antigravity/brain/...`.
- **Юзеркейсы**: Быстрый поиск планов и walkthrough.
- **Тесткейс**: Вывод списка файлов `task.md`, `walkthrough.md`.

---

**Заключение**: Этот каталог покрывает 100% скиллов, доступных в директории `skills`. Теперь у вас есть полный инструментарий для работы в любой сфере: от искусства до системного администрирования.
