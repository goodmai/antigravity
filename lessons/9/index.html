Урок 9: Browser Subagent и “Computer Use”

В этом уроке мы разберем один из самых мощных инструментов Antigravity — встроенный браузер и управляющий им Browser Subagent.

🤖 Что такое Browser Subagent?

Browser Subagent в Antigravity — это не просто окно просмотра, а полноценный агент, использующий технологию “Computer Use”. Он управляет экземпляром Chrome через протокол DevTools и использует мультимодальные модели (Gemini 2.5/3) для визуального восприятия страницы.

Это позволяет агенту “видеть” интерфейс так же, как человек.

Ключевые возможности:

  1. Визуальная навигация: Кликает по элементам, даже если у них нет явных ID, ориентируясь на визуальные метки (“синяя кнопка ‘Купить’”).
  2. Сложные взаимодействия: Заполняет формы, прокручивает списки, использует drag-and-drop.
  3. Визуальная дефектовка: Обнаруживает баги, которые не ловят юнит-тесты (наложение текста, “поехавшая” верстка).
  4. DevTools Integration: Читает Console и Network логи для диагностики ошибок фронтенда (404, JS errors).

🛠️ Как это работает

Агент взаимодействует с браузером через триаду поверхностей:

  1. DOM: Структурное понимание страницы.
  2. Vision: Скриншоты для анализа внешнего вида.
  3. DevTools Protocol: Прямое управление браузером.

📂 Примеры использования

1. Визуальная верификация (Visual QA)

Вы можете попросить: “Проверь, как выглядит форма регистрации на мобильном разрешении”. Агент переключит эмуляцию устройства, сделает скриншоты и предоставит отчет о том, не перекрывает ли кнопка поле ввода.

2. E2E Тестирование “без кода”

Вместо написания скриптов вы говорите: “Зайди на сайт, положи товар в корзину и дойди до оплаты”. Агент выполнит это как пользователь.

3. Отладка фронтенда

Если кнопка не работает, агент может открыть DevTools, увидеть ошибку в консоли и предложить фикс для React-компонента.

📝 Задание

Попробуйте следующую задачу для агента:

“Открой главную страницу документации React, найди раздел про хуки и сделай скриншот примера использования useState.”

Вы увидите, как агент открывает браузер (синяя рамка активности), скроллит страницу и делает снимок нужной области.

Browser Subagent превращает Antigravity из редактора кода в систему, способную взаимодействовать с внешним миром веб-приложений на уровне человека.
