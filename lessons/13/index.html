Урок 13: Облачная разработка (Google Cloud & Firebase)

Antigravity обеспечивает бесшовную интеграцию с облаком через специализированные MCP-серверы (см. Урок 10).

🔥 Firebase MCP

Агент имеет прямой контроль над Firebase проектами. Возможности:

  • firebase init: Инициализация проектов.
  • Firestore Rules: Агент пишет и проверяет правила безопасности лучше человека.
  • Deploy: Деплой функций и хостинга одной командой.
  • Logs: Анализ ошибок в реальном времени.

☁️ Google Cloud Platform (GCP)

Агент управляет ресурсами GCP через Terraform или gcloud CLI, но делает это безопасно.

  • No API Keys: Используется IAM и Identity Federation. Никаких JSON-ключей в коде!
  • Инфраструктура как код: Агент генерирует конфигурации для Cloud Run и GKE.

📝 Задание

Если у вас есть доступ к Firebase проекту:

  1. Попросите агента: “Покажи список моих приложений в текущем Firebase проекте”.
  2. Если проекта нет, попросите “Объясни, как создать новый проект Firebase через консоль”.
