# Урок 16: Мобильная разработка

Antigravity — отличный напарник для мобильной разработки, поддерживающий как Flutter, так и React Native.

---

## 💙 Flutter

Благодаря "родству" с Google, поддержка Dart работает идеально.

- **Генерация UI**: Агент пишет виджеты по скриншотам макетов.
- **State Management**: Реализует логику на Provider, Riverpod или BLoC.
- **Hot Reload Loop**: Агент следит за логами `flutter run` и правит ошибки "на лету".

---

## ⚛️ React Native & Expo

Для React Native агент предпочитает **Expo**.

- Агент берет на себя сложную настройку окружения.
- Навигация через Expo Router.
- Интеграция нативных модулей.

Кейс: Разработчики создают MVP приложений за один вечер, используя Antigravity для кода и Expo Go для теста на телефоне.
