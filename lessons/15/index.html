Урок 15: CI/CD Pipeline Generation

Автоматизация доставки кода — финальный аккорд инженерии. Antigravity умеет генерировать пайплайны для GitHub Actions, GitLab CI и Cloud Build. Тесно связано с Уроком 12 (Тесты) и Уроком 14 (Docker).

🔁 Генерация пайплайнов

Агент знает синтаксис YAML и лучшие практики CI/CD.

Сценарий: “Настрой CI пайплайн для проверки PR”. Агент создаст .github/workflows/pr-check.yml с шагами:

  1. Checkout: Клонирование репозитория.
  2. Setup: Установка Node.js/Python/Go с кэшированием зависимостей.
  3. Lint & Test: Запуск проверок.
  4. Reporting: Отправка отчетов о покрытии (Codecov).

🔐 Keyless Security

Для Google Cloud Build агент настроит Workload Identity Federation. Это современный стандарт безопасности, который позволяет аутентифицироваться в облаке без хранения долгоживущих ключей JSON в секретах GitHub.

📝 Задание

  1. Попросите агента: “Создай GitHub Action для запуска тестов при каждом пуше в ветку main”.
  2. Изучите созданный файл .github/workflows.
