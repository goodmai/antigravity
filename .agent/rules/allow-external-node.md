# Правило доступа к внешней директории

Агенту разрешено взаимодействовать с файлами по пути: `~/node` (или абсолютный путь, например, `/Users/username/node`).

## Инструкции для модели:
1.  **Scope Expansion:** При поиске определений или модулей, если они не найдены в локальном `node_modules`, проверять директорию `~/node`.
2.  **Path Resolution:** Всегда интерпретировать символ `~` как домашнюю директорию пользователя.
3.  **Read Permission:** Разрешено выполнять команды `ls` и `cat` (или аналогичные инструменты чтения) для файлов внутри этой директории.
