# QA Skill Tester

This skill acts as a **Quality Assurance Architect**. It tests _other_ skills by generating validation artifacts and running regression suites.

## Features

1.  **Orchestration**:
    - Uses `run_qa.py` to drive the entire testing lifecycle.
    - Integrates with `prompt-optimizer` (Qwen/Gemini personas).

2.  **Artifact Generation**:
    - **Test Plans**: Plan & User Cases (in Markdown) generated by Gemini using the `Test Architect` persona.
    - **Checklists**: Excel (`.xlsx`) checklists generated via `xlsx` skill logic (includes Frontend Checks: Readability, Clickability, Overlaps).
    - **Bug Reports**: Word (`.docx`) reports generated via `docx` skill logic.

3.  **Validation**:
    - **Qwen Lead**: Validates outputs against quality standards.
    - **UI Checks**: Specific focus on frontend regressions (overlaps, broken links).

4.  **Archival**:
    - Automatically moves artifacts to `artifacts/<skill_name>/test/<timestamp>/`.
    - Uses the `artifact-archival` workflow.

## Usage

### 1. Optimize Prompts (Setup)

Before running, ensure prompts are tuned:

```bash
python3 skills/qa-skill-tester/scripts/optimize_qa_prompts.py
```

### 2. Run QA Cycle

Run the tester against a specific skill:

```bash
skills/qa-skill-tester/.venv/bin/python3 skills/qa-skill-tester/scripts/run_qa.py --skill path/to/skill
```

### 3. Generate Lab Materials

Generate the Pitch Deck for Lab 2:

```bash
skills/qa-skill-tester/.venv/bin/python3 skills/qa-skill-tester/scripts/create_lab2_materials.py
```
